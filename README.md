# Data Analytics Project

## Overview

This project demonstrates an end-to-end data analytics workflow, from raw data processing to business insights visualization. The project includes data loading and analysis using Python, data cleaning, SQL-based querying in PostgreSQL, dashboard development in Power BI, and the creation of a final report and presentation.

The objective is to transform raw data into meaningful insights that support data-driven decision-making.

---

## Dataset

* Source: customer_shopping_behavior.xls
* Format: CSV
* Records: 3900 rows
* Features: 18 columns

The dataset contains information related to Customer Shopping Behaviour Analysis.

## Project Workflow

### 1. Data Loading

* Imported dataset into Python.
* Performed initial data inspection.
* Verified data types and structure.

### 2. Exploratory Data Analysis (EDA)

* Generated summary statistics.
* Identified trends and patterns.
* Analyzed distributions and relationships between variables.
* Created visualizations for deeper understanding.

### 3. Data Cleaning

* Handled missing values.
* Removed duplicate records.
* Corrected inconsistent data formats.
* Treated outliers where necessary.

### 4. SQL Analysis (PostgreSQL)

* Loaded cleaned data into PostgreSQL.
* Wrote SQL queries to answer key business questions.
* Performed aggregations, filtering, joins, and ranking operations.

### 5. Dashboard Development

* Connected PostgreSQL data to Power BI.
* Built interactive dashboards.
* Created KPIs, charts, and filters for analysis.

### 6. Reporting & Presentation

* Prepared a detailed analytical report.
* Summarized findings and recommendations.
* Created a presentation using Gamma AI for stakeholder communication.

## How to Run

### Prerequisites

* Python 3.x
* PostgreSQL
* Power BI Desktop

### Steps

1. Clone the repository:

```bash
git clone https://github.com/your-username/project-name.git
```

2. Install required Python packages:

```bash
pip install -r requirements.txt
```

3. Load the dataset into PostgreSQL.

4. Run the SQL queries from the `sql/` folder.

5. Execute the Python notebook/script for EDA and data cleaning.

6. Open the Power BI dashboard file (`.pbix`) and refresh the data connection.

7. Review the final report and presentation for project findings.
